## Project Overview

The main idea of this final project is to explain and re create the filtered based algorithm on how applications like Netflix, Hulu or HBOMax shows each person different movies and shows based on a recommendation systems that takes the cosine and vectors of similarity depending on their genres and keywords.

The database used for this project was taken from Kaggel - The Movies Dataset
